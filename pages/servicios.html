Servicios

¡La propuesta que tengo para vos es cumplir con lo que soñás para tu evento!

Para ello, solemos coordinar un primer encuentro - virtual o presencial - con el cliente, para poder considerar qué opciones tenemos que se adecúen a tus expectativas y a tu presupuesto.

Nuestros servicios incluyen:

  • Armado de CandyBar o Mesa Dulce personalizada, con packaging diseñado especialmente para tu evento.
  • Armado para el momento de la torta principal o el brindis.
  • Armado de cortina Shimmer o Stand de glitter.
  • Set de Carteles de luz de neón.
  • Set de Luz violeta, ideal para las fiestas temáticas Blancas o Fluorescentes.
  • Impresión de Banners fotográficos.

¡Y muchas opciones más!
